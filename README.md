# Tech Stock Analysis

This repository contains a semester portfolio analysis project focusing on three major technology stocks: Apple Inc. (AAPL), Microsoft Corporation (MSFT), and Alphabet Inc. (GOOGL).

## Objectives
- Gather comprehensive stock information and financial statements.
- Retrieve historical price data for the calendar year 2024.
- Calculate annual returns based on closing prices.
- Identify major institutional investors for Apple.
- Document findings and analysis for ongoing research.

The detailed analysis is in `analysis-summary.md`.
